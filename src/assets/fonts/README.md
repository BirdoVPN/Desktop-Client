# Inter Font Files

Download the variable font files from https://github.com/rsms/inter/releases
and place them here:

- `Inter-Variable.woff2` — Regular (required)
- `Inter-Variable-Italic.woff2` — Italic (optional)

These are imported by `src/styles/fonts.css` and referenced via `@font-face`.
